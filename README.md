# ContactCore-Python-OOP-Foundations
# Project 06: Basic OOP Contact Model

## Overview
ContactCore is a basic project that shows the shift from functional programming to Object-Oriented Programming (OOP). This project combines contact information and management behaviors in a single, reusable `Contact` class.

## The Blueprint
This project represents a real-world contact using three main aspects of OOP:
1. **Attributes:** Storing state (`name`, `email`, `phone_number`).
2. **Methods:** Defining actions (`display_info`, `update_email`).
3. **Instantiation:** Creating unique objects from one template.

## Technical Specifications
- **Language:** Python 3.x
- **Focus:** Class Definition and Encapsulation
- **Key Component:** The `__init__` dunder method for object initialization.
- **Data Structure:** List-based object collection for batch processing.
